# E8 Visual Lab

A learning project for building intuition around vectors, projections, symmetry, and eventually the E8 lattice.

## v0.1.0 - Vector Playground

This version visualizes basic 2D vector addition using Python, NumPy, and Matplotlib.

## What I learned 

Vectors represent direction and magnitude. By adding vectors together, we can model movement, forces, projections, and higher-dimensional structures.

This is the first foundation for understanding the E8 lattice.
